Overview

This project is a React-based Discussion Forum Application built using Dicoding Forum API.
The development is carried out in two integrated phases, where Phase 2 is a direct continuation of Phase 1.

Phase 1: Core application development (features, architecture, UI/UX)
Phase 2: Testing, CI/CD, and production deployment


Phase 1 – Core Forum Application

Developed a fully functional forum web application with features:

User authentication (register & login)
Thread listing & detail view (with comments)
Create thread & comment
State management using Redux
Modular, scalable architecture with clean separation between UI & state
ESLint & React best practices implementation

Goal: Build a functional, maintainable, and user-friendly forum application.


Phase 2 – Testing, CI/CD & Deployment

Enhanced the application to production level by adding:

Automation Testing (Unit, Integration, End-to-End)
CI/CD Pipeline using GitHub Actions
Auto Deployment via Vercel
Branch Protection & Quality Control
Integration of React ecosystem tools

Goal: Ensure the application is reliable, tested, and continuously deployable.

Project Relationship

Phase 2 is a direct continuation of Phase 1, focusing on:

Improving code quality through testing
Automating build, test, and deployment process
Transitioning from development → production-ready application


Tech Stack
Frontend: React, Redux
Testing: Jest, React Testing Library, E2E Testing
CI/CD: GitHub Actions
Deployment: Vercel
Linting: ESLint


Key Highlights
End-to-end React application development
Clean and scalable state management (Redux)
Full testing implementation (Unit → E2E)
CI/CD pipeline & automated deployment
Production-ready web application
